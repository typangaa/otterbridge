# v0.3 plan — narrow weir to a pure stdio-cli agent orchestrator

**Status:** approved (design Q&A 2026-06-16). Breaking release.

## Decision

weir orchestrates **only** local CLI agents via `stdio-cli`. It is neither an
HTTP client (no calling out to `/v1/chat/completions`) nor an HTTP server (no
accepting MCP/API calls over a port). The single MCP transport is stdio.

Rationale: the actual deployment uses 100% CLI-agent backends (agy, claude,
hermes, gemini), each of which owns its own auth and network access. Dropping
the HTTP client + server surface yields a smaller, dependency-light binary with
no network attack surface — fully consistent with the "single binary, key-free,
CLI-native" positioning.

`BackendKind` stays an enum with a single `StdioCli` variant: keeps every
existing `type = "stdio-cli"` config valid and leaves room for a future
MCP-client backend type without another breaking change.

## Removals

**Code**
- delete `src/backends/openai_compat.rs`; drop `pub mod openai_compat;`
- `config/mod.rs`: drop `BackendKind::OpenaiCompat`; drop `ServerConfig.transport`
  + `.port` + `default_transport()` + `default_port()` + Default fields
- `error.rs`: drop `Http(#[from] reqwest::Error)` variant
- `Cargo.toml`: drop `reqwest` (and its rustls TLS features)
- `cli/backend.rs`: drop `add_backend_openai`, the OpenaiCompat arms in
  `backend_to_json` / `backend_summary` / `build_and_health`
- `main.rs`: drop `BackendAddOpenai`, the `Openai` subcommand, its dispatch arm,
  the `use OpenaiCompatBackend`, and the factory arm
- `server/tools.rs`: drop the `use` + factory arm + summary arm
- `cli/status.rs`: drop `transport`/`port` server props and `base_url`/`model`
  openai props; backend `type` enum → `["stdio-cli"]`

**Docs / examples**
- CLAUDE.md, README.md, DESIGN.md, weir.example.toml: drop openai-compat +
  HTTP-transport content; reframe as CLI-only
- delete `examples/providers/{ollama,llamacpp}.toml`
- rewrite `examples/use-cases/{local-only,full-stack}.toml` around stdio-cli
- keep `examples/providers/{claude-code,gemini,openrouter}.toml` (all CLI)
